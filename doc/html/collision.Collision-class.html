CircleCollidedWithCircle(circ1, circ2)
Static Method

source code 

Identifica colisão entre Retângulo entre Circulo e Retângulo.

Passos:

1) Calcula a distância D entre o centro dos circulos com o Teorema de Pitagoras.

2) Se calcula a soma S dos radios dos circulos.

3) Se D <= S significa que os circulos se chocaram.

Parameters:
  • circ1 - Circulo
  • circ2 - Retângulo
Returns:
True se colidiu, caso contrário False
